# Android蓝牙工具

## 功能

* 封装经典蓝牙、低功耗蓝牙BLE
* 检索：不过滤检索、过滤名字检索、过滤Mac地址检索
* UUID：不过滤、过滤
* 通讯：读取、写入蓝牙信息
